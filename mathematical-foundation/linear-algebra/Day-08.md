---

# What is a Norm?

A norm is a mathematical function that measures the magnitude (length) of a vector.

Example:

```
v = [3, 4]
```

The norm tells us how "large" this vector is.

---

# Types of Norms

## 1. L1 Norm (Manhattan Norm)

The L1 norm is the sum of the absolute values of all vector components.

Measures:

* Total distance traveled along each axis.

Applications:

* Feature selection
* Sparse machine learning models
* Lasso Regression

---

## 2. L2 Norm (Euclidean Norm)

The L2 norm measures the straight-line distance from the origin.

Measures:

* Actual geometric length of the vector.

Applications:

* Machine Learning
* Deep Learning
* Gradient Descent
* Embeddings
* Euclidean Distance

---

## 3. L∞ Norm (Maximum Norm)

The L∞ norm is the largest absolute value among all vector components.

Measures:

* The largest component of the vector.

Applications:

* Error bounds
* Optimization
* Constraint checking

---

# Why Norms Matter in AI

Norms are used throughout AI systems.

Examples:

* Measuring model errors
* Computing distances between vectors
* Normalizing embeddings
* Optimization algorithms
* Gradient calculations

Without norms, AI models cannot properly compare or optimize numerical representations.

---

# Real-World AI Connection

When embeddings are generated by Large Language Models, norms help measure vector magnitudes and are often used before computing similarity scores.

Norms are also used during neural network training to control model weights and improve numerical stability.
