# Python Concepts Showcase

## Project Overview

This project is a lightweight Python showcase meant to express core programming knowledge without turning into a full product. It demonstrates object-oriented design, decorators, generators, JSON storage, modular structure, and basic error handling using only the Python standard library.

## What This Project Shows

- A TaskManager class for object-oriented task handling
- A logging decorator with functools.wraps
- A generator function that yields tasks one at a time
- JSON-based persistence with safe file handling
- Simple, readable module organization
- A small demonstration workflow that runs when the script is executed

## Folder Structure

- main.py: script entry point for the concept demo
- task_manager.py: task logic and class-based operations
- decorators.py: reusable logging decorator
- generators.py: generator-based iteration example
- storage.py: JSON load and save helpers
- tasks.json: example storage file
- requirements.txt: dependency list

## Technologies Used

- Python 3
- Standard library only
- JSON, datetime, pathlib, and typing modules

## Python Concepts Demonstrated

- Classes and methods
- Decorators
- Generators
- try/except/else/finally error handling
- Module-based design
- File persistence

## How to Run

From the project root, run:

```bash
python main.py
```

## Example Output

The script will print a short demonstration of task creation, viewing, searching, updating, deleting, and saving.

## Learning Outcome

This repository is intended as a personal expression of Python skills: organized, readable, and focused on demonstrating good software engineering habits.
